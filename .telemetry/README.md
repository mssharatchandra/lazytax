# .telemetry/

This folder contains the source-of-truth product telemetry artifacts for
LazyTax: what the product is, what is currently tracked, what may be tracked,
and the privacy rules that govern implementation.

| File | Purpose |
|---|---|
| `product.md` | Product, value, entity, and integration model |
| `current-state.yaml` | Factual inventory of tracking in the codebase |
| `tracking-plan.yaml` | Approved target event contract |
| `delta.md` | Current-to-target implementation backlog |

Tax documents and calculated tax outcomes must never be added to this folder.
